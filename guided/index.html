Guided Sorting (software simulation)

Items are presented in a fixed order. You select the set at the start, and the item number advances automatically.

Please read this first. An item goes to Sorting if it has any contamination OR any disruptors. If it has neither, it goes to No Sorting.

Disruptors

Any item with buttons, zips, trims, or non-textile attachments.

  • Buttons, zips
  • Studs, rivets, snaps, eyelets
  • Sequins / embellishment
  • Non-textile attachments (plastic/metal parts)

Contamination

Items with moisture, oil, or a large stain. (No harmful substances used.)

  • Wet/damp fabric
  • Oil / grease marks
  • Large food/soil stain

How to move through the trial

Before answering each item, press Start Item to begin that item. After answering both questions, press Next Item to save the response and move on.

You must press Start Item for every new item, and then press Next Item to continue to the next one.
