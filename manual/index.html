Manual Sorting (software simulation)

Hold the physical garment. When the trial starts, click the bin you think the item belongs in.

Participant briefing

Please read this first. An item goes to Sorting if it has any contamination OR any disruptors. If it has neither, it goes to No Sorting.

Disruptors

Any item with buttons, zips, or non-textile attachments.

  • Buttons, zips
  • Studs, rivets, snaps, eyelets
  • Non-textile attachments (plastic/metal parts)
Button
Zip
Eyelet

Contamination

Items with moisture, oil, or a large stain. (No harmful substances used.)

  • Wet/damp fabric
  • Oil / grease marks
  • Large food/soil stain

How to move through the trial

Before each item, press Start Item to begin that item.

After selecting a bin, press Next Item to save the response and move to the next garment.

How to use this simulation

When the trial starts, you will see two bins. Click the correct one.
